# QA Reports

This folder contains per-sprint quality assurance reports.

## Naming Convention
```
qa-sprint-NNN-phase-name.md
```

## Report Template

Each QA report should include:

```
## Sprint / Phase
## Date
## What Was Tested
## Commands Run + Output Summary
## Passed
## Failed
## Visual / UI Issues
## Responsiveness Issues
## Accessibility Issues
## Console Errors
## Build Errors
## Missing States
## Suggested Fixes
## Sign-off
```
