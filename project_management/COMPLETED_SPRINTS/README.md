# Completed Sprints

This folder contains archived sprint files, named by completion date and sprint number.

## Naming Convention
```
sprint-NNN-short-description.md
```

## Examples
```
sprint-001-project-management-bootstrap.md
sprint-002-portfolio-content-scaffolding.md
```

Sprints are moved here when all acceptance criteria are met and `CURRENT_SPRINT.md` has been updated to the next sprint.
